# Internal Employee Portal

Employee-facing internal portal. Handles daily time tracking, end-of-shift reporting, leave requests, outage reporting, and internal ticketing (HR/Finance/IT).

## Rules Reference

Detailed specs live in `.claude/rules/`:

- `stack.md` — tech stack and tooling
- `structure.md` — project folder structure
- `commands.md` — available pnpm scripts
- `conventions.md` — coding conventions
- `layout.md` — three-column dashboard layout
- `features.md` — feature specifications
- `database.md` — Prisma schema conventions and models
- `components.md` — component patterns (modals, accordions, emoji picker, FAB)
