# Dashboard Layout

## Three-Column Layout

The main dashboard is a fixed three-column layout. All primary features live on this single page.

### Left Sidebar (Time Tracking)
- Current date display
- Live clock (updates every second) with AM/PM indicator
- Timezone selector dropdown (default: PHT — Philippine Standard Time)
- Time In / Time Out button (toggles based on state)
- Two side-by-side buttons below:
  - **Report Outage** — opens outage report modal
  - **Leave Request** — opens PTO request modal

### Center Column (Shift Report)
- Always visible, no click to reveal
- **Mood picker:** "How are you feeling right now?" with 5 emoji options:
  - Excited, Happy, Neutral, Tired, Stressed
  - Single select, clickable emoji cards
- **Three text areas:**
  1. "What are your accomplishments today?" (required, itemized)
  2. "What were your challenges? What are your next steps?" (required, itemized)
  3. "How can the company best support you at this time?" (optional, management-only visibility)
- Submit button
- **Report History** below the form — accordion list of past reports

### Right Sidebar (Info)
- **US Holidays** — scrollable list with holiday name and date, collapsible "See less/See more"
- **About Us** — two tabs:
  - Core Values (Trustworthy, Driven, Customer-Obsessed)
  - Purpose (Mission + Vision)
- **FAQ** — link list (Club Portal, Bulletin, Payroll FAQ Sheet, Product Request, Referral Form)

## Floating Elements
- **Ticketing button** — floating action button for filing tickets (HR/Finance/IT concerns)
